My Projects

BackTap - Control your phone with taps on the back

A simple "port" of an accessibility feature seen on iOS 14 and Android 11, where the user can double or triple tap on the back of their phones to invoke certain actions. This application brings that feature to older Android versions. It detects taps on the back of a user's phone using accelerometer data and then executes the user's selected options.


A* Pathfinding Visualization

A simple A* pathfinding algorithm visualizer. You can set the start, end, and wall nodes all through the graphical interface and the program will highlight the squares to go in purple. This was built with Python 3 and the interace was made using tkinter.


JavaScript Graphing Calculator

A simple, easy-to-use graphing calculator implemented in JavaScript. It can handle graphing rational functions, first and second derivatives, calculating relative extrema and points of inflection, and can do calculations for the fundamental theorem of calculus.


MusiTapp - Knock-based Media Control

An easy-to-use tool to control your Android phone's media output by knocking or tapping on the table the phone rests on. By using values from the accelerometer, we can easily determine if the user single, double or triple knocks on the table to play/pause music, go to next track, or go to previous track respectively.


NutriScan - Food Consumption Tracker

An easy-to-use Android app + Web interface to keep track of one's caloric intake. The app makes use of the Microsoft Azure Cognitive Vision API and it's OCR function to read a product's nutrition label and parse it to keep track of calories, fat, protein, and carbohydrate intake. The web interface accesses a Firebase Database to sync data between your phone and the web.
